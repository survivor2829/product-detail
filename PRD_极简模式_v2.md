# 小玺AI 极简模式 需求文档 v2

> 生成日期：2026-04-10
> 状态：初版（v2 更新：加入模板选择 + 三栏布局）

## 一句话定义
把现有复杂表单替换为三栏极简界面：左栏输入 → 中间实时预览 → 右栏选模板，所见即所得。

## 目标用户
1. 公司内部同事（熟悉产品，追求效率）
2. 客户/经销商（不太懂产品，需要最低学习成本）

## 核心功能（必须实现）

### 1. 三栏布局（参考黑米AI）
页面分为三个并排区域，不再是上下堆叠的长表单：

- **左栏 — 输入区**（约300px宽）
  - 品类标签切换（设备类/配耗类/耗材类/工具类）
  - 上传图片区：产品主图（必填，自动抠图）、场景底图（可选）、Logo（可选）、二维码（可选）
  - 文案粘贴框：一个大文本框，粘贴产品资料
  - "一键生成详情页"按钮
  
- **中间 — 预览区**（主体区域，自适应宽度）
  - 实时展示生成的详情页长图
  - 每个模块右上角有操作按钮：编辑 / 重新生成 / 显隐开关
  - 模块之间可拖拽排序
  - 底部有"导出高清PNG"按钮

- **右栏 — 模板选择区**（约250px宽）
  - 以缩略图网格展示所有可用模板
  - 当前选中的模板高亮显示
  - 点击切换模板 → 中间预览区实时更新为新模板风格（保留用户数据不变）

### 2. 风格模板系统（8-10套）
每套模板定义一整套详情页的视觉风格，包括：
- 主色调 + 辅助色 + 背景色
- 字体风格（标题/正文）
- 模块样式（圆角/阴影/间距/背景纹理等）
- 模块默认组合顺序

**模板切换逻辑：**
- 切换模板只改变视觉风格，不改变用户已填的数据和AI生成的文案
- 切换后中间预览区实时刷新
- 每个品类有一个默认推荐模板，用户也可以选其他任何模板

### 3. 一键生成（AI全自动）
- 用户在左栏上传图片 + 粘贴文案 + 选模板（可不选，用默认）
- 点击"一键生成" → AI完成全部工作：
  - 解析产品参数
  - 生成营销文案（卖点、故事、VS对比等）
  - 根据品类选择合适的模块组合
  - 用选定模板的风格渲染
- 结果直接出现在中间预览区

### 4. 模块级编辑（在预览页上直接改）
- 每个模块右上角三个按钮：
  - ✏️ **编辑**：弹出精简编辑面板，可改文案和图片
  - 🔄 **重新生成**：AI重新生成该模块文案（其他模块不变）
  - 👁 **显隐开关**：隐藏不想要的模块
- 编辑后实时刷新预览，不用重新提交

### 5. 模块拖拽排序
- 模块之间可拖拽调整上下顺序
- 拖完后预览立即更新
- 导出时按调整后的顺序

### 6. 导出PNG
- 保留现有Playwright截图导出
- 只导出可见（未隐藏）的模块
- 按用户排列的顺序导出

### 7. 四品类全支持
- 设备类、配耗类、耗材类、工具类
- 在左栏顶部标签切换，不跳转页面
- 每个品类的AI提示词、模块组合沿用现有配置

## 可选功能（后续迭代）
1. 历史记录：保存之前生成的详情页，可回看和重新编辑
2. 批量生成：一次传多个产品资料，批量出详情页
3. 一键翻译：生成英文版详情页（跨境场景）
4. 模块级模板：同一个模块有多种布局样式可选
5. 自定义模板：用户可以保存自己调整过的风格为新模板

## 明确排除（不做的事）
1. ~~旧表单页面（build_form.html）~~：彻底删除
2. ~~自由画布编辑器~~：不做Canva那种自由拖拽，只做模块级结构化编辑
3. ~~AI生成场景图~~：不接入AI绘图，场景图仍由用户上传或使用内置图片
4. ~~多语言界面~~：界面只支持中文，不做国际化

## 技术可行性备注
- [2026-04-10] 条件：现有Flask + Jinja2架构。三栏布局需要重写前端页面，但后端API可复用。模板系统可通过CSS变量 + 模板配置JSON实现，不需要重写HTML模块。
- [2026-04-10] 条件：模板切换实时预览。可以用CSS自定义属性（CSS Variables）控制颜色/字体/圆角等，切换模板只需替换一组CSS变量，不用重新渲染HTML。
- [2026-04-10] 条件：拖拽排序。引入SortableJS（CDN），前端拖拽后更新模块顺序，导出时按新顺序渲染。
- [2026-04-10] 条件：单模块AI重新生成。需要新增API端点，接收模块类型+已有产品数据，只返回该模块的文案。
- [2026-04-10] 条件：8套模板。每套模板本质是一组CSS变量+配色方案，工作量主要在设计阶段，代码实现量不大。

## 用户流程
```
1. 打开首页 → 看到三栏布局
2. 左栏：选品类标签 → 上传产品图 → 粘贴文案
3. 右栏：浏览模板缩略图 → 点选喜欢的风格（可选，有默认）
4. 点"一键生成" → 等待30秒
5. 中间预览区出现完整详情页
6. 想改文案 → 点模块的"编辑"按钮
7. 想让AI重写 → 点模块的"重新生成"
8. 不要的模块 → 点"隐藏"
9. 调顺序 → 拖拽模块
10. 想换风格 → 右栏点另一个模板，预览立即切换
11. 满意 → 点"导出高清PNG"
```

- [ ] 右栏有8套模板缩略图，点击可切换，预览区实时更新风格
- [ ] 切换模板不丢失用户数据和AI生成的文案
- [ ] 每个模块有编辑/重新生成/显隐三个操作按钮
- [ ] 编辑文案后预览区3秒内刷新
- [ ] 模块可拖拽排序，拖完预览立即更新
- [ ] 隐藏的模块不出现在导出PNG中
- [ ] 四个品类都走通完整流程
- [ ] 旧build_form.html已移除
